# ✨ Egi's Personal Showcase & Portfolio Website

An interactive, responsive landing page and developer showcase website. Designed with modern layout configurations, smooth animations, and curated color palettes to showcase games, web projects, and technical skills.

## 🚀 Features

- **Premium Design Aesthetics**: Vibrant color schemes, glassmorphism, and responsive CSS grids.
- **Interactive Portfolio Grid**: Dynamic showcase cards featuring hover animations.
- **Vanilla JavaScript Logic**: Smooth transitions, mobile menu toggles, and tab routing.
- **Responsive Layout**: Optimized for perfect presentation on desktop, tablet, and mobile browsers.

## 🛠️ Technologies Used

- **HTML5** - Semantic layout structure
- **Vanilla CSS3** - Custom styling, flexbox/grid, and micro-animations
- **JavaScript (ES6)** - Client-side animations and DOM control
